# phishing-email-analysis-dhl-impersonation
This project shows a basic phishing email analysis done as SOC Level 1 practice.

#Scenrio
An email was received pretending to be from DHL asking for payment to release a delivery.

#What I checked
- Sender email and domain
- Message content and urgency sign
- Suspicious URL structure
- Indicator of phishing
  
# Result 
The email was identified as a phishing attempt based on domain mismatch and social engineering indicators.

#Skilled Practiced
- Email analysis
- Phishing detection
